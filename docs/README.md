# B1SLayer Documentation

This folder contains detailed documentation for B1SLayer, a lightweight SAP Business One Service Layer client for .NET.

## Available Documentation

- **[SAP Community Blog Post](B1SLayer-Post-Sap-Community.md)** - Original blog post from SAP Community covering all features and usage patterns of B1SLayer, including:
  - Getting started and setup
  - Performing basic and advanced requests
  - Session management
  - Specialized requests (attachments, batch operations, ping)
  - Best practices and examples
